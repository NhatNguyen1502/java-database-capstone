# User Stories

## Admin User Stories
- As an Admin, I want to manage user accounts so that I can control who has access to the system.
- As an Admin, I want to view a dashboard of system activity so that I can monitor the application's usage.
- As an Admin, I want to manage system configurations so that the application runs smoothly.

## Patient User Stories
- As a Patient, I want to register and log in so that I can access my personal dashboard.
- As a Patient, I want to view available doctors and their schedules so that I can book an appointment.
- As a Patient, I want to book an appointment so that I can receive medical care.
- As a Patient, I want to view my upcoming appointments so that I can remember when to visit the doctor.
- As a Patient, I want to cancel or reschedule an appointment so that I can manage my time effectively.
- As a Patient, I want to view my medical records so that I can keep track of my health history.

## Doctor User Stories
- As a Doctor, I want to log in so that I can access my dashboard.
- As a Doctor, I want to manage my availability schedule so that patients know when I am free.
- As a Doctor, I want to view my upcoming appointments so that I can prepare for patient visits.
- As a Doctor, I want to add prescriptions for patients so that they can get the medication they need.
- As a Doctor, I want to view patient records so that I can provide better diagnosis and treatment.
